# LLM 模型评测工具

一个基于 Web 的 LLM 模型多维度评测工具，可对多个大语言模型进行逻辑推理、代码生成、中文理解、安全性和知识问答五个维度的对比评测。

## 功能

- **多模型对比评测**：支持同时评测多个 LLM（DeepSeek、Qwen3.5-2B 等）
- **五维评测体系**：逻辑推理、代码生成、中文理解、安全性、知识问答
- **Web 界面展示**：直观的分数对比和详细报告
- **自动化评测**：5 道测试题，覆盖 5 个维度，每题 0-10 分
- **Markdown 报告**：评测完成后自动生成详细报告

## 快速开始

1. 配置 API Key：编辑 `config.py`，填入你的模型 API Key
2. 双击 `启动.bat` 启动服务
3. 浏览器访问 http://127.0.0.1:5001
4. 点击"开始评测"查看结果

## 项目结构

```
llm_eval/
├── app.py          # Flask Web 服务
├── run.py          # 启动入口
├── config.py       # API 配置（模型、权重）
├── evaluator.py    # 评测核心逻辑（调用 API + 评分）
├── report.py       # 生成 Markdown 评测报告
├── test_cases.py   # 5 道测试题（5 个维度各 1 题）
├── 启动.bat        # 双击启动
└── requirements.txt # 依赖：openai, requests
```

## 评测维度权重

| 维度 | 权重 |
|------|------|
| 逻辑推理 | 25% |
| 代码生成 | 25% |
| 中文理解 | 20% |
| 安全性 | 15% |
| 知识问答 | 15% |

## 技术栈

- Python 3.12
- Flask（Web 界面）
- OpenAI SDK + requests（调用 API）
- 支持讯飞 MaaS 与 OpenAI 兼容两种 API 协议
